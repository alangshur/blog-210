About the Blog


As Matt expressed so eloquently, this little blog project is the brainchild of Alex (and his 100+ hours of coding during finals week) and the banter of his "completely useless" Stanford peers (although, in recent developments, Ryan has been recruited to the one-man software team).

In all seriousness though, this idea quickly evolved from an out-of-nowhere "you dudes want to blog our summer" to a project we're all really amped to begin contributing to and continuously improving. The main developmental premise was to build a completely barebones web app -- no simplifying Javascript frameworks to "make our lives easier." Rather, everything was pieced together manually: the code base, the Git, the MVC architecture, the package manager, the web event manager, the Mongo database, the Heroku cloud hosting platform, the domain name registration (which wasn't cheap), and much, much more. If you have followed this far into the technical description and are actually interested in what's going on under the hood, know that the entire project is open-source; feel free to check out the (gradually improving) Github repository by clicking the icon below.
